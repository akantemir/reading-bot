# Reading Bot

Telegram bot for tracking daily reading habits with GigaChat integration for motivational messages.

## Features
- Daily 20-minute reading reminders at 20:30 MSK
- Statistics tracking (daily, weekly, monthly)
- GigaChat AI-powered motivational messages
- Mini-goal celebrations (3, 7, 10, 30 consecutive days)

## Commands
- `/start` - Start the bot
- `/statistika` - View statistics with AI motivation
- `/mesyac` - Monthly statistics
- `/rekord` - Record streak

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure environment variables (TELEGRAM_TOKEN, GIGACHAT_AUTH_KEY)
3. Run: `python3 bot.py`
